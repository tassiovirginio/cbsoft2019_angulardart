Talk:Gamifying Software Testing
Writing good software tests is difficult and not every developerâ€™s favourite occupation. If an activity is so difficult, boring, or otherwise unattractive that people do not want to engage with it, then gamification offers a solution: By turning the activity into a fun and competitive task, participants engage, compete, and excel. In this talk, I will explore how this idea can be applied to software testing. Our ongoing work with the Code Defenders game demonstrates that players engage with testing, and perceive it as a fun activity. At the same time, by participating in the game, players produce test suites that are far superior to anything automated testing tools generate. This illustrates the potential of using gamification to address some of the many problems that we are facing today in software testing. There are, however, many challenges ahead, and I will outline some of the challenges and research opportunities related to gamifying software testing.
